# Corggle

Corggle は、ユーザーがチャットを通じて様々なシーンを体験できるアプリケーションです。このプロジェクトは Flutter を使用して構築されています。

## 特徴

- ユーザー認証（Google サインイン）
- チャット機能
- チャット履歴の表示
- ユーザープロファイルの管理

## インストール

このプロジェクトをローカル環境にインストールするには、以下の手順に従ってください。

1. リポジトリをクローンします。

   ```sh
   git clone https://github.com/yourusername/iroiro.git
   cd iroiro
   cd corggle-app
   ```

2. 必要な依存関係をインストールします。

   ```sh
   flutter pub get
   ```

3. Firebase プロジェクトを設定し、`google-services.json`（Android）および`GoogleService-Info.plist`（iOS）を適切なディレクトリに配置します。

4. アプリを実行します。

   ```sh
   flutter run
   ```
